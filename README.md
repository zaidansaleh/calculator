# calculator

A simple REPL-based calculator for evaluating arithmetic expression.

## Usage

Run `make` to build the `calculator` binary:

```sh
make
./calculator
```

Example session:

```sh
> 0.1 + (2e1 - 3e-1) * 4.0 / 5
0.02
```
